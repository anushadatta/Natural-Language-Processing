To run the model, run the app.py file.

PREREQUISITES
Ensure Python3 and pip is downloaded and added to system environment variables.

The list of packages required are as follows: nltk, numpy, sklearn, pandas, json, random, re.

Please ensure they are downloaded through the command !pip install <package_name>

INSTRUCTIONS TO RUN CODE
- Keep the dataset and the app.py in the same directory.
- Run the app.py file in a python editor or in a command line by typing: python <filename>.py
